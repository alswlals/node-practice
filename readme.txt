1. 패키지

2. 의존성
    - 라이브러리 자체를 의존성이라 부름
    - 개발하는 프로젝트(애플리케이션, 라이브러리)에서 설치하고 사용하는 패키지
    - 일반 의존성 : 개발하고 있는 프로젝트에서 사용하는 패키지로 꼭 빌드와 배포에 포함되어야 한다.
    - 개발 의존성 : 개발에 필요하거나 도움이 되는 패키지로 빌드와 배포에 포함하지 않는다.
